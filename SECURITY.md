# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.1.1   | :white_check_mark: |
| 4.1.0   | :x:                |
| 4.0.x   | :x: |
| < 4.0.0 | :x:                |

## Reporting a Vulnerability

To report a vulnerability, either open an issue or send an email to i-Vis Research Lab ivis@cs.bilkent.edu.tr.
